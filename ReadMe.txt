This project satisfies the requiremnts for Assignment4 Part2

Node.js needs to be installed for this to function.
storage for server is hosted on mongoLab

open the node.js command prompt and navigate to this folder
run "npm install"
	this will install dependencies need to run server
once installed start the server using the command "node ." or "node server"

the server will be hosted on localport 8080

To view documentation and all API calls go to http://localhost:8080/ in your browser
	all api calls are described under default, press "default" or "Expand Operations" to see them
	use "Try it out!" buttons to issue API calls

There is some functionality within the browser, to view this go to http://localhost:8080/api in your browser

Server uses: Node.js, mongoose, expressjs, and Jade.
